# 📡 CRC Simulation
Προσομοίωση του αλγορίθμου **Cyclic Redundancy Check (CRC)** σε Java. Το πρόγραμμα δημιουργεί τυχαία δυαδικά μηνύματα, υπολογίζει το CRC (FCS), προσομοιώνει τη μετάδοση μέσω θορυβώδους καναλιού με δεδομένο **Bit Error Rate (BER)** και εμφανίζει στατιστικά σχετικά με την ανίχνευση σφαλμάτων.

## ✨ Χαρακτηριστικά <a name="features"></a>
- Δημιουργία τυχαίων δυαδικών μηνυμάτων.
- Εισαγωγή από τον χρήστη του μήκους του μηνύματος (**k**), του **BER** και του πολυωνύμου **P**.
- Υπολογισμός του **CRC (FCS)** με διαίρεση modulo-2 (XOR).
- Προσομοίωση μετάδοσης μέσω θορυβώδους καναλιού.
- Έλεγχος της εγκυρότητας του ληφθέντος πλαισίου.
- Υπολογισμός και εμφάνιση στατιστικών αποτελεσμάτων.

## 🚀 Χρήση <a name="usage"></a>
Ο χρήστης εισάγει:
- το μήκος του μηνύματος (**k**),
- το πολυώνυμο **P**,
- το **Bit Error Rate (BER)**.

Στη συνέχεια δημιουργούνται τυχαία μηνύματα, υπολογίζεται το CRC, προσομοιώνεται η μετάδοσή τους μέσω θορυβώδους καναλιού και εμφανίζονται τα στατιστικά αποτελέσματα.

## 💾 Εγκατάσταση <a name="installation"></a>

Ακολούθησε τα παρακάτω βήματα για να κάνεις clone και να τρέξεις το project locally. Υποστηρίζεται σε **Linux, macOS και Windows**.

### 1. Clone το repository

Βεβαιώσου ότι έχεις εγκατεστημένο το Git και τρέξε:

```bash
git clone https://github.com/XM-George/CRC-Simulation.git
cd CRC-Simulation

```

### 2. Άνοιξε το project
Μπορείς να ανοίξεις το project σε
- IntelliJ IDEA(συνιστάται)
- Eclipse
- οποιοδήποτε άλλο Java IDE

Βεβαιώσου ότι το **src** folder είναι ορισμένο ως source root αν χρειάζεται.

### 3. Τρέξε την εφαρμογή

#### Αν χρησιμοποιείς IDE:

Πήγαινε στο:

```bash

src/Main.java

```

και τρέξε απευθείας την **Main** class.

#### Αλλιώς αν χρησιμοποιείς το terminal

```bash

cd src
javac Main.java
java Main

```
